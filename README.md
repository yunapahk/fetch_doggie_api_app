# Fetch Doggie
A website that fetches any dog breed providing an image and origin.

# Technologies Used
* HTML
* CSS
* JavaScript
* jQuery
* Bootsrap/Bootwatch
* RapidAPI
* [CodePen Animation](https://codepen.io/stivaliserna/pen/jObPyKe)
